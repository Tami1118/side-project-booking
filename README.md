## 專案技術
- Vite 環境建置
- Vue 3 Composition API
- Tailwind + SCSS 響應式設計
- TypeScript
- Pinia 狀態管理
- Vue Router 路由管理


## 專案套件
- Vue Axios 串接 RESTful API 資料
- VCalendar 日期套件
- Vee Validate 表單驗證管理
- Sweet Alert 2 彈跳視窗設計
- Swiper 圖片輪播設計
