## 主要功能
- 會員註冊登入
- 房型介紹
- 線上訂房
- 訂房紀錄
- 房型管理
- 訂單管理


## Src 資料夾說明
- assets - 圖片及SCSS樣式
- components - Vue 組件
- data - 資料庫
- interfaces - TypeScript interface
- mixins - 共用函式
- plugins - 套件
- router - vue router
- stores - 狀態管理
- views - Vue 網站頁面


## 專案技術
- Vite 環境建置
- Vue 3 Composition API
- Tailwind + SCSS 響應式設計
- TypeScript
- Pinia 狀態管理
- Vue Router 路由管理


## 專案套件
- Vue Axios 串接 RESTful API 資料
- VCalendar 日期套件
- Vee Validate 表單驗證管理
- Sweet Alert 2 彈跳視窗設計
- Swiper 圖片輪播設計
- AOS Animate 動畫套件
